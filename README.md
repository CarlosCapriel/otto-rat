# Otto Rat 👾🎮
Otto Rat 🎮 es un juego interactivo 2D para fortalecer las habilidades matemáticas 🧮 .

Hecho como proyecto escolar siguiendo la metodología SCRUM 📋 y utilizando tecnologías y herramientas como:
- Git/GitHub 🐈‍⬛
- GodotEngine 🤖
- GDScript 💻
